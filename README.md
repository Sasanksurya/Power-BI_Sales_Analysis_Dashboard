# Power-BI_Sales_Analysis_Dashboard
Power BI dashboard for analyzing sales trends, product performance, and business insights.

## Project Overview
This project presents an interactive Power BI dashboard developed to analyze sales performance and identify key business insights.

- Sales trend analysis
- Product performance analysis
- Regional sales comparison
- Interactive filters for better analysis

## Tools Used

- Data Visualization
- Data Analysis
